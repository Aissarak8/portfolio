# Getting Started with AI Agents & Automation

I'll be honest up front: **I'm a beginner** with AI agents. This post is less "here's how it works" and more "here's what I'm learning and why." I'm writing it partly to organize my own understanding.

## The terms, in plain language

- **AI Agent** — an AI that doesn't just answer once, but can *plan steps, use tools, and act* toward a goal.
- **RAG (Retrieval-Augmented Generation)** — giving an AI access to your own documents so it can answer using real, specific information instead of guessing.
- **MCP (Model Context Protocol)** — a standard way to connect AI models to tools and data sources.
- **n8n** — a visual automation tool for connecting apps and building workflows (with or without AI).

## My learning plan

1. **Start simple** — build a basic chatbot and understand prompts.
2. **Add retrieval** — try a small RAG project so the AI can answer from documents.
3. **Automate something real** — use n8n to remove a repetitive task.
4. **Combine them** — my planned *AI WhatsApp Agent* project ties these ideas together.

## Why I'm excited

Automation and AI are where I want to build my career as an **AI Engineer**. Starting small, shipping projects, and writing about them is how I plan to get there.

More posts to come as I actually build these. 🤖
